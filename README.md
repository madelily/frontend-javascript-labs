# frontend-javascript-labs

> 🎓 Full JavaScript Learning Map  
> **1일 30분 · 1~5 커밋 · 개념 누락 없는 JS 학습 기록 레포**

이 레포는  
**JavaScript를 “처음부터 끝까지, 사고 단위로 쪼개서” 다시 학습**하기 위한 개인 학습 저장소입니다.

- 강의 요약 ❌  
- 코드 복붙 ❌  
- 하루 몰아치기 ❌  

👉 **개념 하나 = 파일 하나 = 커밋 하나**  
👉 **매일 30분, 짧지만 끊기지 않는 학습**을 목표로 합니다.

---

## 📌 학습 규칙 (중요)

- ⏱ 하루 학습 시간: **30분**
- 🔢 하루 커밋 수: **1 ~ 5 커밋**
- 🧩 커밋 기준: **“하나의 사고 / 개념 단위”**
- 📁 폴더는 미리 만들어두되,  
  **README·코드 파일은 ‘공부한 날’에만 생성**
- 📈 커밋 히스토리 자체가 학습 타임라인이 되도록 유지

## 🧠 학습 방식

### 1️⃣ Step 단위 학습
- `stepXX-주제명` 폴더 = 하나의 큰 학습 주제
- step 시작일에 `README.md` 생성
- 세부 개념은 **파일 단위로 분리**

### 2️⃣ 파일 네이밍 규칙
01-개념명
02-개념명
03-개념명

👉 파일 하나 = 개념 하나 = 커밋 하나

---

## 📝 Step README 작성 방식 (요약)

각 step의 `README.md`는  
**학습 메모 + 사고 정리용**으로 사용합니다.

- `# / ## / ###` 단계적 정리
- `- [ ] / - [x]` 체크리스트
- `---` 구분선
- “왜?” / “헷갈린 점” 중심 기록
